Roadmap

Upcoming Focus

A snapshot of the near-term plans we are executing before expanding beyond the current platforms.

Fiat on/off ramp

We are integrating the Changelly Fiat API so you can buy or sell TON directly inside the wallet. Identity verification and payment processing remain on the provider’s side, while funds land in the same secure vault.

Open modules

After the DEX adapters stabilize, we will publish them in a dedicated public repository. This will let integrators reuse the modules, file issues, and ship their own improvements with full transparency.

Additional platforms

Native mobile shells—and a browser extension if needed—are planned once the current security and audit checklist is cleared. The same protections and flows will carry over to every device.
